# Guide Utilisateur RédacNews

## Accès

- **URL** : https://redacnews.link
- **Navigateurs supportés** : Chrome, Firefox, Safari, Edge (dernières versions)

## Première connexion

1. Rendez-vous sur https://redacnews.link/login
2. Entrez vos identifiants fournis par l'administrateur
3. Changez votre mot de passe temporaire

## Fonctionnalités

### Conducteur
Le conducteur permet de planifier le déroulé d'une émission minute par minute.
- Créez un nouveau conducteur pour chaque émission
- Ajoutez des éléments : sujets, sons, jingles, pauses pub
- Réorganisez par glisser-déposer
- Le timer se calcule automatiquement

### Sujets
Rédigez vos sujets avec l'éditeur Google Docs intégré.
- Chaque sujet crée automatiquement un Google Doc
- Collaborez en temps réel avec votre équipe
- Liez des fichiers audio de la médiathèque

### Médiathèque
Stockez et organisez tous vos fichiers audio (interviews, jingles, etc.).
- Upload par glisser-déposer
- Lecture directe dans l'interface
- Organisation par collections
- Recherche dans les titres et descriptions

### Prompteur
Affichez vos scripts en plein écran pour la lecture à l'antenne.
- Défilement automatique réglable
- Contrôle au clavier (espace = pause)
- Mode sombre pour la régie

## Raccourcis clavier

| Raccourci | Action |
|-----------|--------|
| Espace | Pause/lecture du prompteur |
| Flèche haut/bas | Navigation dans le prompteur |
| Ctrl+S | Sauvegarder |
